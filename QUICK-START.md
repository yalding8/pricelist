# 🚀 快速开始 - 顾问表单系统

## 一分钟启动

### 1️⃣ 启动Web服务

在终端执行：

```bash
python3 pricelist-web-app.py
```

### 2️⃣ 打开浏览器

访问：**http://localhost:5001**

### 3️⃣ 填写表单

按照界面提示填写：
- 房源信息（名称、户型、租金、地址、租期）
- 房东优惠（可选，点击"➕添加"）
- 异乡补贴（可选，点击"➕添加"）
- 赠送礼包（点击卡片勾选）
- 顾问信息（你的姓名、电话、微信）

### 4️⃣ 生成报价单

点击 **"🚀 生成报价单"**

### 5️⃣ 下载文件

生成成功后：
- 📄 下载HTML（网页版）
- 🖼️ 下载PNG（微信分享图片）

---

## 📱 发送给客户

### 微信分享（推荐）

1. 找到下载的PNG图片
2. 在微信聊天中点击 [+] → [相册]
3. 选择PNG图片发送
4. 客户可以点击放大查看

### 其他方式

- 邮件：附件发送HTML或PNG
- WhatsApp：直接发送图片
- 网盘：上传HTML供客户下载
